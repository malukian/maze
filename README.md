# maze
FRA UAS BSRN Alternative 7
Maze Game Python

This python script simulates the maze game. User has an opportunity to choose the size of the labyrinth and enjoy playing in free time.

This  program  was developed as part of the examination for 'Operating Systems and Computer Networks' modules of the IBIS/EBIS 
(International/Engineering Business Information Systems) degree program at Frankfurt University of Applied Sciences in 2021.
